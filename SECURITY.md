# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | ✅ |
| < 0.2   | ❌ |

## Reporting a Vulnerability

Report vulnerabilities by opening a GitHub Issue at
https://github.com/Lento47/arcana/issues.
Do not include exploit code in public reports — describe the issue and
we will follow up for details.

We aim to acknowledge reports within 48 hours.
